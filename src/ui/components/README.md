# `src/ui/components/` — Reusable UI Components

## Purpose

Contains small, reusable Streamlit widgets used across multiple pages.

## Components

| Component | Description |
|-----------|-------------|
| `report_card.py` | Renders a triage result as a visual card with all details |
| `severity_badge.py` | Colour-coded pill showing P1/P2/P3/P4 severity level |
